# mango公众管理系统

#### 介绍
市面上见过有太多的后台管理系统，但是简约快速开发的暂时没有发现，本框架旨在解决快速开发。

后端使用技术：Springboot+SpringSecurity+Jwt 数据库：MySQL+Redis 

前端：Vue+axios。

后期后逐步达到一个完善的管理系统
