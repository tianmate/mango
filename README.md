# mango公众号运营管理系统

#### 介绍
市面上见过有太多的后台管理系统，但是针对微信公众号开发几运营的暂时没有发现，本框架旨在解决微信公众号快速开发。

后端使用技术：Springboot+SpringSecurity+Jwt 数据库：MySQL+Redis 

前端：Vue+axios。另外集成了腾讯云和阿里云短信服务，集成微信公众号中的接口，包含扫码登录、发消息、回复消息、点击菜单等事件处理等。

后期后逐步达到一个完善的公众号运营管理系统
